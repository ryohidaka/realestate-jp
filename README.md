不動産関連のユーティリティ関数を提供するライブラリ

## インストール

### Node.js

```sh
npm i realestate-jp
```

### Deno

#### Add Package

```sh
deno add jsr:@japanese/realestate
```

#### Import symbol

```sh
import * as realestate from "@japanese/realestate";
```
